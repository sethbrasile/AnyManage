## Progress Tracking Convention

Use this checkbox format to track task status:

- `[ ]` = Todo (not started)
- `[-]` = In Progress (add date when started)
- `[x]` = Completed (add date when finished)

**Example:**
```
- [ ] Upcoming task
- [-] Working on this now - Started 2024-01-15
- [x] Finished this task - Done 2024-01-10
```
